¿Cómo probarlo?

Hacer "npm init -y", luego "npm install axios dotenv express express-session nodemon passport passport-google-oauth20" y correr el servidor con "node server.js"
Acceder a http://localhost:3000/profile
Iniciar sesión
De hacerlo bien, serás redirigido a otra página que dirá que iniciaste exitosamente
Regresa a http://localhost:3000/profile para mostrar tu información de usuario y otro mensaje
Cierra la sesión en http://localhost:3000/profile
